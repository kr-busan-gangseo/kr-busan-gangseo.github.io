# 강서구 · 부산광역시

> 강서구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-busan-gangseo.github.io/**

강서구는 낙동강이 남해로 흘러들며 쌓아 올린 삼각주와 김해평야, 그리고 남해 끝자락의 가덕도까지 아우르는 부산에서 가장 넓은 자치구입니다. 면적은 181.5㎢로 부산 전체의 약 23.6%를 차지하지만, 그 상당 부분이 대저도·맥도·명지도 같은 강 가운데 섬과 갯벌·습지로 이루어진 점이 다른 도심 구들과 뚜렷이 구별됩니다. 대저 토마토와 명지 대파, 강동 깻잎이 자라는 비옥한 농경지가 펼쳐지는 한편, 녹산국가산업단지와 부산신항, 김해국제공항이 자리 잡아 농업·물류·첨단 제조가 한 지역에 공존합니다. 명지국제신도시와 에코델타시티 개발로 새 아파트촌이 빠르게 들어서면서, 전통적인 강 하구 농어촌에서 서부산의 성장 거점으로 성격이 바뀌고 있는 곳입니다.
